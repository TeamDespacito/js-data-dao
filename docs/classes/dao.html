Class DAO<T>

Foi projetado a classe para ser operada como classe generica para ser aplicavel em qualquer classe de persistencia, de forma montar as seguintes operacoes

  • buscar todos
  • buscar por id
  • inserir
  • alterar
  • deletar
  • fazer uma busca paginada

os metodos buscar todos ( findAll ) e query paginada (paginatedQuery) utilizam do sistema de sintaxe de query de busca do js-data, para mais detalhes de como utilizar, entre no seguinte link:

http://www.js-data.io/docs/query-syntax
